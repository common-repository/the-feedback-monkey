A plugin to show TheFeedbackMonkey widgets on your WordPress theme.

== Description ==

Show your business reviews from TheFeedbackMonkey, choose between multiple reviewbox types with a built in admin-dashboard. Enable or disable widgets at anytime or customize widgets. This plugin uses the service from TheFeedbackMonkey.com and requires you to have an account to be able to use this plugin.

## How to use

### API Key
Insert your API key into the 'settings' page in the `FBM Admin`

### Settings
On the settings page you are able to set default options for classes with the name `monkey-review`

### Adding a new custom reviewblock
In the Add Reviewbox page you are able to upload your own custom reviewbox which will show in the custom reviewboxes page once you submit.

### Applying them on your WordPress Page
You can insert the `<div>` tag anywhere on your page. Go to:
1. Appearance
2. Widgets
3. Custom HTML
4. Paste `div` tag

## Errors?
